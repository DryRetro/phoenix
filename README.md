## Installation 
1 - Updating system
```bash
sudo apt update
sudo apt install build-essential
```

2 - Run this command to install libs and enter activation code
```bash
make new
```

3 - Run to prepare wallets.json file with random User Agents and set up proxy to specific profile
```bash
make init
```

4 - Run System
```bash
make run
```
